# cormorant-example
This is an example of what [cormorant](http://www.github.com/adrienthiery/cormorant)
allows you to do.

To start, just :
 * ```bower install```
 * ```git submodule init && git submodule update```
 * (install cormorant : ```cd cormorant && bower install```
 * link to your webserver !
